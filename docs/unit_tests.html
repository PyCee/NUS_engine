Unit Tests are simple programs designed to test specific functions of the NUS library. These programs may rely on code originally tested in other unit tests, or may be independant of other unit tests.
3d_math
a program that computes various pieces of 3d mathematics and prints out results
source
basic_window
opens a window and detects input. window can be closed by pressing the esc key or clicking the close window button that is usually in the top-left or top-right (dependant on your operating system)
source
gpu_detection
utilizes vulkan to detect, and then print, information regarding the clients graphical processing unit. This program will open a window to get some information and automatically close said window once the information has been obtained.
source
vulkan_clear
similar to basic_window, this program opens a window and can be closed by the esc key or the close button. vulkan is utilized to make a gpu clear the window to a solid color
source
hello_triangle
a program that build upon vulkan_clear and uses shaders to draw a solid colored triangle, then display it on a window
