NUS_library is a group of functions and structures to abstract away some of the more complicated tasks of programming a 3d video game in the C programming language.
This library is being actively developed by Kim Pampusch. Going by the name of "PyCee" on github, Kim is a computer science student at St. Paul college that programs the NUS library with the goal of developing 3D games.
