NUS library utilizes a smaller library, named NUS-Models (NUSM), for storing 3D model data.
Documentation for NUSM can be found in PDF form. (ongoing development)
NUSM uses a custom binary file format used to store 3D models, which may be read using a function from the NUSM library.

In addition to the functions to manage a NUSM file in an application, NUSM supplies a command-line utility that will read a 3D model file (of any type supported by Assimp) and create a corresponding NUSM file.
The command can be run as follows:

NUSM <path to source file> <path to desired file output>
(paths are relative to working directory)

> NUSM cube.dae cube.nusm
