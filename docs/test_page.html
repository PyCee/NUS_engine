Debian

Copy the followingasd into a terminal:
cd ~
git clone https://github.com/PyCee/NUS_library.git
cd NUS_library/
make install
make
Alternativly, to enable vulkan validation layers, you may replace
make install
with
make debug

Dependencies

Vulkan

  • To run a program that uses NUS Library, the computer must have vulkan-compatible gpu drivers installed
  • To compile a program utilizing NUS Library, the VulkanSDK must be installed and setup
