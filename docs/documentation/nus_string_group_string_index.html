NAME

nus_string_group_string_index - obtains index of a string

C SPECIFICATION

NUS_result nus_string_group_string_index( NUS_string_group string_group, const char *string, unsigned int *p_index);

PARAMETERS

  • string_group is a string group that an index will be obtained from
  • string is the string that will be searched for
  • p_index is a pointer to an int that will return the index

RETURN VALUE

On Success
  • returns NUS_SUCCESS
On Failure
  • returns NUS_FAILURE
  • string could not be found

DETAILS

  • string_group must have been initialized or built
  • string must be a null-terminated c-string
  • p_index must be a pointer to an allocated int

EXAMPLE

Code:
NUS_string_group string_group; unsigned int index; nus_string_group_build(&string_group, "string!!!"); nus_string_group_string_index(string_group, "string!!!", &index); printf("The index of the string is: %d\n", index);
Output:
The index of the string is: 0
