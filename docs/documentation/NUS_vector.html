NAME

NUS_vector - a 3d vector struct

C SPECIFICATION

typedef struct NUS_vector{
double x, y, z;
} NUS_vector;

PARAMETERS

  • x is where the vector points along the x-axis
  • y is where the vector points along the y-axis
  • z is where the vector points along the z-axis

RETURN VALUE

returns a NUS_vector struct populated with the passed floats
