NAME

nus_string_group_copy - builds a copy of a string group

C SPECIFICATION

void nus_string_group_copy( NUS_string_group *p_string_group_dest, NUS_string_group string_group_src);

PARAMETERS

  • p_string_group_dest is a pointer to a string group
  • string_group_src a string group. It's strings will be appended to p_string_group_dest

RETURN VALUE

does not return a value

DETAILS

  • p_string_group_dest must have been initialized or built
  • string_group_src must have been initialized or built

EXAMPLE

Code:
NUS_string_group string_group_0; NUS_string_group string_group_1; nus_string_group_build(&string_group_0, "this is originally in string_group_0"); nus_string_group_init(&string_group_1); nus_string_group_copy(&string_group_1, string_group_0); nus_string_group_print(string_group_1);
Output:
NUS_string_group contains 1 string(s): string 0: this is originally in string_group_0
