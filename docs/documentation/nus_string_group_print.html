NAME

nus_string_group_print - prints string group values

C SPECIFICATION

void nus_string_group_print(NUS_string_group NUS_string_group_);

PARAMETERS

  • NUS_string_group_ is the string group that will have it's information printed out

RETURN VALUE

does not return a value

DETAILS

  • no details of note

EXAMPLE

Code:
NUS_string_group string_group; char *test = "test"; string_group.strings = malloc(strlen(test) + 1) string_group.count = 1; strcpy(string_group.strings, test); nus_string_group_print(string_group);
Output:
NUS_string_group contains 1 string(s): string 0: test
