NAME

nus_vector_add - subtracts a vector from another vector

C SPECIFICATION

NUS_vector nus_vector_subtract( NUS_vector NUS_vector_0, NUS_vector NUS_vector_1);

PARAMETERS

  • NUS_vector_0 is the first vector
  • NUS_vector_1 is subtracted from the first vector

RETURN VALUE

returns the first vector - the second vector

DETAILS

  • no details of note

EXAMPLE

Code:
NUS_vector test_vector_0 = nus_vector_build(5.0, 1.0, 0.0); NUS_vector test_vector_1 = nus_vector_build(2.0, 1.0, 3.0); NUS_vector result = nus_vector_subtract(test_vector_0, test_vector_1); nus_vector_print(result);
Output:
{3.0, 0.0, -3.0}
