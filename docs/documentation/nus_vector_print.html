NAME

nus_window_print - prints vector values

C SPECIFICATION

void nus_window_print(
NUS_vector NUS_vector_);

PARAMETERS

  • NUS_vector_ is the vector which the printed information will originate

RETURN VALUE

does not return a value

DETAILS

  • no details of note

EXAMPLE

Code: 

	  NUS_vector test_vector = {
	  .x = 1.0,
	  .y = 0.0,
	  .z = -1.0
	  };
	  nus_vector_print(test_vector);
Output: 

	  {1.0, 0.0, -1.0}
