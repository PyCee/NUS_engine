NAME

nus_string_group_set - resets an already allocated element of a string group

C SPECIFICATION

NUS_result nus_string_group_set( NUS_string_group *p_string_group, unsigned int index, const char * const string);

PARAMETERS

  • p_string_group is a pointer to a string group that will have a string changed
  • index tells what string in p_string_group is to be reset
  • string is the c-string that will be set

RETURN VALUE

On Success
  • returns NUS_SUCCESS
On Failure
  • returns NUS_FAILURE
  • index was not valid

DETAILS

  • p_string_group_dest must have been initialized or built
  • index must be less than the count of strings in p_string_group
  • string must be a null-terminated c-string

EXAMPLE

Code:
NUS_string_group string_group; nus_string_group_build(&string_group, "a string"); nus_string_group_set(&string_group, 0, "a new string"); nus_string_group_print(string_group);
Output:
NUS_string_group contains 1 string(s): string 0: a new string
