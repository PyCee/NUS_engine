NAME

nus_vector_cross - computes the cross product of 2 vectors

C SPECIFICATION

NUS_vector nus_vector_cross( NUS_vector NUS_vector_0, NUS_vector NUS_vector_1);

PARAMETERS

  • NUS_vector_0 is the first argument to the cross product
  • NUS_vector_1 is the second argument to the cross product

RETURN VALUE

returns the cross product of the 2 vectors

DETAILS

  • both parameters must have a magnnitude greater than 0.0
  • the cross product is a vector prependicular to both of the input vectors
  • the cross product has length = magnitude of first vector * magnitude of second vector * sin of angle between the 2 input vectors
  • there are 2 vector perpendicular to both inputs. The result is determined by the order of the input (see link below)
  • Cross Product

EXAMPLE

Code:
NUS_vector test_vector_0 = nus_vector_build(0.0, 0.0, -1.0); NUS_vector test_vector_1 = nus_vector_build(-1.0, 0.0, 0.0); NUS_vector result = nus_vector_cross(test_vector_0, test_vector_1); nus_vector_print(result);
Output:
{0.0, 1.0, 0.0}
