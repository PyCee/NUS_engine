NAME

nus_vector_interpolate - interpolates between 2 vectors

C SPECIFICATION

NUS_vector nus_vector_interpolate(
NUS_vector NUS_vector_0,
NUS_vector NUS_vector_1,
double t);

PARAMETERS

  • NUS_vector_0 is the vector to interpolate from
  • NUS_vector_1 is the vector to interpolate too
  • t determines the interpolation between the 2 vectors

RETURN VALUE

returns a vector interpolated between the 2 vectors by t

DETAILS

  • a t <= 0.0 will return NUS_vector_0
  • a t >= 1.0 will return NUS_vector_1
