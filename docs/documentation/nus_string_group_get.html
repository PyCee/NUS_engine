NAME

nus_string_group_get - obtains string from string group at specific index

C SPECIFICATION

NUS_result nus_string_group_get( NUS_string_group string_group, unsigned int index, char **result);

PARAMETERS

  • string_group is a string group that a string will be obtained from
  • index is the index of the desired string
  • result is a pointer to a c-string that will be set

RETURN VALUE

On Success
  • returns NUS_SUCCESS
On Failure
  • returns NUS_FAILURE
  • index was not valid

DETAILS

  • string_group must have been initialized or built
  • index must be less than the count of strings in string_group
  • string should not be allocated

EXAMPLE

Code:
NUS_string_group string_group; char *recieved_string; nus_string_group_build(&string_group, "a string"); nus_string_group_get(string_group, 0, &recieved_string); printf("The recieved string is: %s\n", recieved_string);
Output:
The recieved string is: a string
