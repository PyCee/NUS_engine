NAME

nus_vector_dot - computes the dot product of 2 vectors

C SPECIFICATION

double nus_vector_dot(
NUS_vector NUS_vector_0,
NUS_vector NUS_vector_1);

PARAMETERS

  • NUS_vector_0 is the first argument to the dot product
  • NUS_vector_1 is the second argument to the dot product

RETURN VALUE

returns the dot product of the 2 vectors
