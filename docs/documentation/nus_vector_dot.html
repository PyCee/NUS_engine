NAME

nus_vector_dot - computes the dot product of 2 vectors

C SPECIFICATION

double nus_vector_dot( NUS_vector NUS_vector_0, NUS_vector NUS_vector_1);

PARAMETERS

  • NUS_vector_0 is the first argument to the dot product
  • NUS_vector_1 is the second argument to the dot product

RETURN VALUE

returns the dot product of the 2 vectors

DETAILS

  • Dot Product
  • The Dot Product is equal to: magnitude of the first vector * magnitude of the second vector * cosine of the angle between the 2 vectors

EXAMPLE

Code:
NUS_vector test_vector_0 = nus_vector_build(0.0, 0.0, 1.0); NUS_vector test_vector_1 = nus_vector_build(0.7071, 0.0, 0.7071); double result = nus_vector_dot(test_vector_0, test_vector_1); printf("dot product = %f\n", result);
Output:
dot product = 0.7071
