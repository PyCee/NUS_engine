NAME

nus_string_group_free - frees the memory allocated in a string group

C SPECIFICATION

void nus_string_group_free(NUS_string_group *p_string_group);

PARAMETERS

  • p_string_group is a pointer to the string group that will have it's memory freed

RETURN VALUE

does not return a value

DETAILS

  • p_string_group must have been initialized or built

EXAMPLE

Code:
NUS_string_group string_group; nus_string_group_build(&string_group, "this string will be allocated"); nus_string_group_print(string_group); nus_string_group_free(&string_group); printf("string_group has been freed\n"); nus_string_group_print(string_group);
Output:
NUS_string_group contains 1 string(s): string 0: this string will be allocated string_group has been freed NUS_string_group contains 0 string(s):
