NAME

nus_vector_cmp - tests if 2 vectors are within a certain range of eachother

C SPECIFICATION

NUS_vector nus_vector_cmp(
NUS_vector NUS_vector_0,
NUS_vector NUS_vector_1
double range);

PARAMETERS

  • NUS_vector_0 is the first vector of the comparison
  • NUS_vector_1 is the second vector of the comparison
  • range is the value the distance will be compared to

RETURN VALUE

If the distance between the 2 vectors < range
  • returns 1
If the distance between the 2 vectors >= range
  • returns 0

DETAILS

  • the range should be positive, else there is no chance for a result of 1
  • the range should be more than 0 unless the vectors are constant, as floting-point number comparisons can be tricky
