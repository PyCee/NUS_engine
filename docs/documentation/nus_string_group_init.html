NAME

nus_string_group_init - initializes a string group

C SPECIFICATION

void nus_string_group_init(NUS_string_group *p_string_group);

PARAMETERS

  • p_string_group is a pointer to the string group that will be initialized

RETURN VALUE

does not return a value

DETAILS

  • p_string_group should not have been initialized or built before

EXAMPLE

Code:
NUS_string_group string_group; nus_string_group_init(&string_group); nus_string_group_print(string_group);
Output:
NUS_string_group contains 0 string(s):
