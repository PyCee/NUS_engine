NAME

nus_window_build - creates and displays a window

C SPECIFICATION

NUS_result nus_window_build(
char *title,
unsigned short width,
unsigned short height,
NUS_window *p_NUS_window_);

PARAMETERS

  • title is a sring that will appear in the title bar of the created window
  • width is the starting horizontal dimension for window
  • height is the starting vertical dimension for window
  • p_NUS_window_ tells where in memory to create the window

RETURN VALUE

On Success
  • returns NUS_SUCCESS
On Failure
  • returns NUS_FAILURE

DETAILS

  • title must be a null-terminated c-string
  • title should be a unique name to the application
  • width must be the starting horizontal dimension for window
  • height must the starting vertical dimension for window
  • p_NUS_window_ must be a pointer to an allocated block of memory of size NUS_window
  • a window must be created before user event handling
  • a window must be created before creation of a presentation surface
  • this funciton is os-agnostic, with all os-specific code enclosed in os-specific macros
