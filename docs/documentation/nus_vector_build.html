NAME

nus_vector_build - builds a 3d vector

C SPECIFICATION

NUS_vector nus_vector_build( double x, double y, double z);

PARAMETERS

  • x is the x value of the built vector
  • y is the y value of the built vector
  • z is the z value of the built vector

RETURN VALUE

returns a vector built with the 3 parameters as members

DETAILS

  • no details of note

EXAMPLE

Code:
NUS_vector test_vector = nus_vector_build(1.0, 0.0, -1.0); nus_vector_print(test_vector);
Output:
{1.0, 0.0, -1.0}
