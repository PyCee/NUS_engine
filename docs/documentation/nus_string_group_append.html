NAME

nus_string_group_append - appends a c-string to a string group

C SPECIFICATION

void nus_string_group_append( NUS_string_group *p_string_group, const char *string);

PARAMETERS

  • p_string_group is a pointer to the string group that will be appended
  • string is a c-string that will be copied, with the copy being appended to the string group

RETURN VALUE

does not return a value

DETAILS

  • p_string_group must have been built or initialized
  • string must be null-terminated

EXAMPLE

Code:
NUS_string_group string_group; nus_string_group_init(&string_group); nus_string_group_append(&string_group, "appended"); nus_string_group_print(string_group);
Output:
NUS_string_group contains 1 string(s): string 0: appended
