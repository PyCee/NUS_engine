NAME

nus_vector_scale - multiplies a vector by a scalar

C SPECIFICATION

NUS_vector nus_vector_scale( NUS_vector NUS_vector_, double scale);

PARAMETERS

  • NUS_vector_ is the vector to be multiplied by the scalar
  • scale is the scalar value to be multiplied

RETURN VALUE

returns the vector scaled by the scalar

DETAILS

  • no details of note

EXAMPLE

Code:
NUS_vector test_vector = nus_vector_build(0.0, 1.0, 3.5); double scale = 2.0; NUS_vector result = nus_vector_scale(test_vector, scale); nus_vector_print(result);
Output:
{0.0, 2.0, 7.0}
