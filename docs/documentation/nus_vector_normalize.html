NAME

nus_vector_normalize - normalizes a vector

C SPECIFICATION

NUS_vector nus_vector_normalize( NUS_vector NUS_vector_);

PARAMETERS

  • NUS_vector_ is the vector to be normalized

RETURN VALUE

returns the input vector, normalized

DETAILS

  • the input vector must have a magnitude greater than 0.0
  • creates a Unit Vector

EXAMPLE

Code:
NUS_vector test_vector = nus_vector_build(1.0, 1.0, 0.0); NUS_vector result = nus_vector_normalize(test_vector); nus_vector_print(result);
Output:
{0.7071, 0.7071, 0.0}
